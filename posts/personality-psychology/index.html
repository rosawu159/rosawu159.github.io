人格心理學的菜鳥筆記

發表於2023-09-03|更新於2023-09-04|Trends
|閱讀量:

因緣際會下需要準備自我介紹，出社會一久發現除了學歷工作外好像沒什麼能說的，頂多講講興趣和喜歡吃什麼(?)。

一邊想自己的個性一邊又有點逃避現實的因素下，開始思考什麼是個性、什麼是人格特性等等的問題。

雖然以下都是大學通識課或社科院會講到的題目，但廢廢一枚大學都跑去修涼課了，以下內容就只是稍微隨手整理的一些，關於網路上對於人格心理學的筆記。

「人格」是什麼？

  • 他像什麼、
  • 他如何變成這個樣子
  • 他為何會這麼做
  • 他是藉由什麼而組織成一個整體。

「人格理論」= 說明人格的「結構」、「歷程」與「發展」

在心理學中，研究人格的理論和方法多種多樣，並且經常交叉應用以獲得更全面的理解。

多位心理學家和研究者在不同的時間和背景下發展和提出的以下四種解釋人格的結構、發展和運作方式。

  1. 特質論（Trait Theory）：
    氣質、他人對此人的整體印象、他是一個xxx的人。像是元素構成整體的概念，trait屬於先天的神經迴路、也就是人的原廠設定。特質論主要關注個體的特質或特徵，假設個體的人格特徵是相對穩定和持久的，代表了個體在不同情境下的表現方式，是具有一致性和可預測的反應。主要應用舉例像是五大特質、DISC、九型人格、塔羅牌、人類圖可用於截長補短，例外像小說描述歸納人格特質、推測其行為的依據。

    主要知名學者像是Cattell（人格因素：每個人身上都應該會具備某16種特質，只是表現有程度上差異; 個人行動取決於：特質 + 角色 + 當下心情）和Allport的人格特質論：
    - 特質是人格的基本單位
    - 人格 = 動力組織 = 統我（Proprium）
    - 特質為行為的可能範疇，根據情境在一定範圍內變動
    - 共同特質(依據文化形態)
    - 個人特質 = 首要特質（只會有一個） + 重要特質 + 次要特質

  2. 類型論（Type Theory）
    「類型」是功能表現，Type Theory假設人格可以分為有限的幾種固定類型或類別。每個類型具有一組獨特的特徵，並且個體屬於某一特定類型。例如，卡爾·榮格提出的人格心理學中包括了多個不同的人格類型，如內向和外向型（ MBTI測驗 ）。了解功能類型，是理解「影響我們判斷和感受的優先要素」是什麼。

其中Trait Theory和Type Theory都類似於給人貼標籤的類型。最大的差異是，Trait Theory將人格特徵視為連續的維度，強調人格特徵的連續性和多樣性，它將人格視為一個連續的光譜，個體可以在各種特質上得分不同。而Type Theory則強調分類和類型的固定性，建立在離散的分類基礎上，例如根據區別個體在特定特質上的表現方式，將人格分為幾個不同的、互不重疊的類型。

  1. 整體論（Holistic Theory）：
    整體論通常強調人格的整體結構和動態。它關注的是人格的統一性和整體性（self = 自我概念），而不僅僅是個別特質或特徵。人有自己的信念、態度和價值，個體的心智和行為不能僅僅通過分析其組成部分來理解，而應該將個體視為一個統一的整體。
    - 知名學者如佛洛伊德的3人格結構：本我、自我、超我
    - 人本學派的Rogers提倡「自我」是有自我實現的動機，這樣的人格成長是無法預測的，擁有樣板只會降低人格的生命力，且創傷會使人格停滯不再成長
    - Bandura的交互決定論
    a. 人類會主動學習，且有能力將內在經驗轉換出來，並一致性的建構成自我系統
    b. 個人事實（認知 = 記憶 + 參與）x 環境 x 行為。也就是說行為不只是內在狀態，也會受到環境影響
    與整體心理學（Gestalt Psychology）相似，這個學派的代表人物包括馬克斯·維爾特海默（Max Wertheimer）、康拉德·朗斯（Kurt Koffka）和沃爾夫岡·科勒（Wolfgang Köhler）。整體心理學強調感知和認知過程中整體結構的重要性，並反對將心理過程細分成獨立的元素。

  2. 建構論（Constructivist Theory）
    建構系統 = 被動社會建構 + 主動個人建構。建構論關注人格是如何通過個體的思想、信仰、價值觀和情感在後天構建的，類似於電腦的自動學習。這種方法強調了個體對自己和世界的解釋和建構過程，因為涉及認知，所以是可以學習和擁有新作為的。George Kelly的Personal Construct Theory可以視為一種建構論方法。
    - 人解釋世界的方式 = 建構系統 = 人格
    - 心理治療 = 心理重建 = 重新建構個人建構系統
    - 強調個人對事件/環境之感受、解釋給予概念化的過程 -> 他採取什麼規則/系統/理論來解釋這件事情
    - 分類事件、紀錄行為過程 -> 能解釋就能預測
    - 心理治療是訓練案主成為更好的預測家：
    - 焦慮：遭遇的事件發生在建構系統以外
    - 害怕：多次的無效建構 -> 懷疑自我、困惑、威脅
    - 威脅：建構系統即將發生重大改變（認知失調、撼動自我認同）-> 會放棄行動
